Executive Summary

Novon Lighting's advanced Boundary-Compensated Linear Testing (BCLT) methodology eliminates Linear Terminal Interference Effect (LTIE) in photometric measurements. Our proprietary Centered Array Photometric Protocol (CAPP) combined with advanced Photometric Dimensional Reduction Protocol (PDRP) and Adaptive Hemisphere Detection Algorithm (AHDA) tests a 560mm LED sample (equivalent to our standard LED board length) within a 1200mm test fixture. This ensures the test sample behaves exactly like a proper continuous section of linear lighting without end plate interference, providing accurate luminous flux data that represents real-world performance in continuous linear installations of any length.

Traditional photometric testing of linear LED luminaires places LEDs along the full length of test fixtures. However, when LEDs are positioned near end plates, their hemispherical light emission is absorbed and incorrectly reflected by these solid boundaries, resulting in artificially reduced luminous flux measurements that do not represent the performance of LEDs in proper continuous installations.

‚úÖ Solution: Boundary-Compensated Linear Testing (BCLT)

Novon Lighting's proprietary Centered Array Photometric Protocol (CAPP) tests a precisely 560mm LED sample within a 1200mm test luminaire. This 560mm test sample corresponds exactly to our standard LED board length and represents a proper continuous section of linear lighting without end plate interference. The larger 1200mm test fixture provides 320mm buffer zones from each end plate, ensuring the 560mm test sample performs identically to how it would in a proper continuous installation.
